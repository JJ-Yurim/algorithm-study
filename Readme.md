## Algorithm Study

알고리즘과 코딩 테스트 문제 풀이를 기록하는 저장소입니다.

프로그래머스, SWEA 등 다양한 플랫폼의 문제를 풀고, 풀이 과정과 사용한 알고리즘을 정리합니다.

### 목표

* 꾸준히 알고리즘 문제 풀기
* 문제 해결 능력 향상
* 다양한 알고리즘과 자료구조 학습
* 풀이 코드와 학습 과정 기록

### 사용 언어

* Python

### 문제 풀이 플랫폼

* Programmers
* SWEA

### 폴더 구조

```text
algorithm-study/
├── programmers/
│   ├── level0/
│   ├── level1/
│   └── level2/
├── swea/
└── README.md
```

### 풀이 기록

각 문제 파일에는 다음 내용을 기록합니다.

* 문제 이름
* 사용한 알고리즘 또는 개념
* 풀이 코드
* 시간 복잡도
* 어려웠던 점 또는 새롭게 배운 점

### Commit Convention

```text
[Programmers] 문제 이름
[SWEA] 문제 이름
[Study] 알고리즘 개념 정리
[Fix] 기존 풀이 수정
```

예시:

```text
[Programmers] 두 수의 합
[Programmers] 문자열 뒤집기
[Study] 완전 탐색 개념 정리
```

### 학습 기록

꾸준히 문제를 해결하며 더 효율적인 풀이 방법을 고민하고 기록합니다.
